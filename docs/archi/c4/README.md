# MindfulSpace – Diagrammes C4

Ces diagrammes complètent les **Architecture Decision Records (ADR)** et
illustrent visuellement l’architecture du projet **MindfulSpace** selon le
modèle **C4** (Context, Containers, Components, Code).

Ils permettent de passer d’une **vision globale du système** à une **description
progressive des composants et des flux métier**, en cohérence avec le cahier des
charges du projet.
