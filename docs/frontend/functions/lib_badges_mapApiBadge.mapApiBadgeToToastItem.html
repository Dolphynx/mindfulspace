    Function mapApiBadgeToToastItem

    • mapApiBadgeToToastItem(apiBadge: unknown): BadgeToastItem

      Convertit un objet brut renvoyé par l'API en structure normalisée BadgeToastItem, directement exploitable par le système de toasts du front.

      Comporte plusieurs sécurités :

      • normalisation de l'ID et du slug en string,
      • fallback cohérent pour titleKey et descriptionKey,
      • conversion systématique de earnedAt vers un ISO string,
      • application d'une icône par défaut (default) si nécessaire.

      Parameters

      • apiBadge: unknown

        Objet brut provenant de l'API.

      Returns BadgeToastItem
